## 命令及其功能
- Help	打开帮助菜单
- Creat	创建数据库
- Open	打开数据库
- Save	保存数据库
- Add row	添加行
- Add col	添加列
- Mod	修改数据
- Locate for	按条件查找
- Delete for	按条件删除
- Clear	清空数据库
